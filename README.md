# HTML Forms Practice

## 📌 Project Overview

This repository contains my HTML Forms practice projects. The main goal of this project is to improve my understanding of HTML form elements, table layouts, and form design by creating different registration and application templates.

## 🚀 Concepts Practiced

### HTML Form Elements

* `<form>`
* `<input>`
* `<label>`
* `<select>`
* `<option>`
* `<textarea>`
* `<button>`
* Form validation using `required`

### Table Concepts

* `<table>`
* `<tr>`
* `<td>`
* `<th>`
* `rowspan`
* `colspan`
* Table alignment and formatting

## 📝 Forms Created

* User Registration Form
* Student Registration Form
* Event Registration Form
* Job Application Form
* Hotel Booking Form
* Community Membership Form
* Survey Form

## 🎯 Learning Objectives

* Understand HTML form structure and user input handling.
* Practice designing forms using tables.
* Improve table formatting and alignment skills.
* Create visually organized form templates.
* Learn the use of labels for better accessibility.

## 🛠️ Technologies Used

* HTML5

## 📚 Key Features

* Multiple registration form templates
* Different table layouts
* Use of labels and input fields
* Dropdown menus and selection controls
* Required field validation
* Organized and beginner-friendly design
